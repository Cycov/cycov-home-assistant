# Add-ons

This directory contains Home Assistant add-ons managed as git subtrees.

## Adding a new add-on subtree

```bash
git subtree add --prefix=addons/<addon-name> <remote-url> <branch> --squash
```

## Updating an existing add-on subtree

```bash
git subtree pull --prefix=addons/<addon-name> <remote-url> <branch> --squash
```
